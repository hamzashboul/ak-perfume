# 🛍️ AK Perfumes

Luxury fragrance e-commerce platform built for the Jordanian market.

🔗 **Live Demo:** [ak-perfume-ten.vercel.app](https://ak-perfume-ten.vercel.app)

---

## ✨ Features

- 🌐 Full AR/EN bilingual support + RTL/LTR auto-switching
- 🛒 Browse, filter & search fragrances
- 🛍️ Add to cart & checkout system
- 💬 WhatsApp integration with pre-filled order messages
- 🎟️ Promo code engine (percentage & fixed discounts)
- 🚚 Smart shipping fees calculated by city
- 🔐 Admin Dashboard — manage products, orders & promo codes
- 📊 Live sales statistics for admin
- 🔒 Supabase Auth + Row Level Security

---

## 🛠️ Tech Stack

- **Frontend:** Next.js 14 · TypeScript · Tailwind CSS
- **State Management:** Zustand
- **Database:** Supabase · PostgreSQL
- **Auth & Security:** Supabase Auth · Row Level Security
- **Deployment:** Vercel
